## 🎯 What is VIA AI?

VIA AI is a **boredom cure app** that generates activities based on your interests. Whether you're into football, cricket, cooking, exercise, or just need something to do — VIA AI has a mission for you.

Built as a single HTML file. No frameworks. No installs. Just open and go.

## ✨ Features

### 🏠 Home
- **10 interest categories** — Football, Cricket, Cooking, Exercise, Sleep, Mind, Photo, Music, Clean, Tech
- **Get Activity** — type any interest and get a tailored mission instantly
- **Boredom Override** — random physical challenge to snap you out of boredom
- **Random Mission** — completely surprise activity from any category
- **🔥 Streak tracker** — tracks your daily activity streak

## 📱 Install as App (PWA)

VIA AI is a **Progressive Web App** — install it like a real app on any device:

**Android (Chrome)**
1. Open the site in Chrome
2. Tap the **"Install VIA AI"** banner at the bottom
3. Tap **Install** — done ✅

**iPhone / iPad (Safari)**
1. Open the site in Safari
2. Tap the **Share button** (□↑)
3. Tap **"Add to Home Screen"**
4. Tap **Add** — done ✅

**Windows / Mac / Linux (Chrome)**
1. Open the site in Chrome
2. Click the **install icon** in the address bar (or tap the banner)
3. Click **Install** — done ✅

> Works **offline** after first install thanks to Service Worker caching.

## 🏅 XP & Level System

- Complete missions to earn **XP**
- Every **100 XP** = **Level Up**
- Levels increase indefinitely
- Daily missions give **bonus +75 XP**
- AI sessions give **+30 XP**
- Timer completions give **bonus XP**

## 🛠️ Tech Stack

- **Pure HTML/CSS/JS** — zero dependencies, zero frameworks
